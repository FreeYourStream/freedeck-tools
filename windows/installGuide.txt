Step 1: Open a powershell or CMD windows
Step 2: enter the command python

go to step 5

Step 3: If the windows store didn't open, type python3
Step 4: Install the program from the Windows store

Step 5: Install the dependencies with
 'pip install -r pip_windows.txt --user'

Step 6: Start the script
 'python auto_page_switcher.py'
